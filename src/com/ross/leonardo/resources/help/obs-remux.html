OBS Remux

What this mode does

Remux changes the container without re-encoding video/audio. That means it is usually very fast and does not reduce quality.

Common use: MKV → MP4 for upload or editing compatibility.

When to use it

  • You recorded in MKV (recommended for safety) and now need an MP4 for upload.
  • Your editor/site prefers MP4 but you don’t want to re-encode.
  • You want a quick “container swap” for compatibility.

How to use it

  1. Select OBS Remux mode.
  2. Choose your input recording (often .mkv).
  3. Choose an output folder.
  4. Click Convert (remux).
  5. Verify the output file plays correctly before deleting your original.

Tip: Keep the original MKV as a backup until you verify the remuxed file.

Notes

  • Remux is not the same as re-encoding—quality should remain the same.
  • If the target container doesn’t support a track/codec, remux may fail.
  • Leonardo prevents overwriting existing files by design.
