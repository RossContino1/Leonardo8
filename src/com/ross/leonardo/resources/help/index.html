Leonardo 10 Help

Leonardo is a lightweight video workflow utility for creators. It uses FFmpeg to prepare video for common pipelines.

Quick Start

  1. Select a Mode (Resolve / OBS Remux / YouTube H.264 / TikTok 9:16).
  2. Choose an Input file (or folder, if supported by your build).
  3. Choose an Output folder.
  4. Click Convert and wait for completion.

Leonardo remembers your last theme, folder, and mode in a configuration file.
